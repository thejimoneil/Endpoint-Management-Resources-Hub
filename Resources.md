## Contributing

We welcome contributions from the community! Here's how you can help:

### How to Contribute

1. **Fork this repository**
2. **Add your resource** to the appropriate section
3. **Submit a pull request** with a clear description of what you're adding
4. **Follow the formatting guidelines** below

### Formatting Guidelines

- Use descriptive link text (not "click here")
- Include a brief description of what the resource offers
- Maintain alphabetical order within sections where appropriate
- Check that links are working before submitting
- For new sections, follow the existing markdown structure

### Suggesting New Sections

If you think a new category would be valuable, open an issue to discuss it before creating a pull request.

### Quality Standards

Resources should:
- Be relevant to Endpoint Management (Intune, ConfigMgr, PowerShell, etc.)
- Provide educational, informational, or networking value
- Be from reputable sources
- Have accurate and working links
