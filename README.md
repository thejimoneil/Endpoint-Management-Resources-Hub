## ü§ù Contributing

We believe in the power of community! Your contributions help make this resource hub more valuable for everyone.

### How to Contribute

1. **Fork this repository** to your GitHub account
2. **Add or update resources** in the appropriate section of [Resources.md](Resources.md)
3. **Submit a pull request** with a clear description of your changes
4. **Wait for review** - we'll review and merge quality contributions promptly

### Contribution Guidelines

#### ‚úÖ What to Include

- **Educational Resources** - Tutorials, guides, documentation, and learning materials
- **Community Content** - Blogs, forums, and expert insights
- **Tools & Scripts** - Practical utilities that solve real problems
- **Official Resources** - Microsoft documentation and official tools
- **Networking Opportunities** - User groups, conferences, webinars, media/podcasts, and community tools

#### üìù Formatting Standards

- Use **descriptive link text** (avoid "click here")
- Include a **brief description** of what the resource offers
- Maintain **alphabetical order** within sections where appropriate
- **Verify links work** before submitting
- Follow the **existing markdown structure** for consistency

#### ‚ùå What Not to Include

- Promotional or commercial content without educational value
- Broken or inactive links
- Resources unrelated to endpoint management
- Duplicate content already in the list

### Suggesting New Sections

Have an idea for a new category? **[Open an issue](https://github.com/thejimoneil/Endpoint-Management-Resources-Hub/issues)** to discuss it with the community before creating a pull request.

### Reporting Issues

Found a broken link or outdated information? **[Report it here](https://github.com/thejimoneil/Endpoint-Management-Resources-Hub/issues)**.
